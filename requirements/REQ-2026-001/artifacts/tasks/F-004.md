mock-claude.sh 行为根据环境变量决定：
- `MOCK_CLAUDE_MODE=success` → 输出固定 markdown 列表，exit 0
- `MOCK_CLAUDE_MODE=fail` → exit 1
- `MOCK_CLAUDE_MODE=empty` → exit 0 + 空输出
- `MOCK_CLAUDE_MODE=slow` → sleep 60 后输出（用于测 watchdog）
