# F-001 · settings.json SessionEnd 注册 + 主 Hook 守门脚本

## 需求描述

在 `.claude/settings.json` 增加 SessionEnd hooks 条目，matcher 为 `prompt_input_exit|logout|other`；新建 `.claude/hooks/extract-experience.sh`，实现同步读 stdin、4 层守门（env/branch/meta/phase）、nohup fork worker、disown，主进程 ≤ 100ms。

## 接口/数据结构

详见 `requirements/REQ-2026-001/artifacts/detailed-design.md` 第 2 节（settings.json）和第 3 节（M2 主 Hook 完整伪代码）。stdin JSON 字段：cwd / transcript_path / hook_event_name / reason / session_id。

## 依赖

### 依赖的 feature

无

### 依赖的模块

- `.claude/settings.json`
- `.claude/hooks/extract-experience.sh`（新建）

## 触及范围

仅 `.claude/settings.json` 和 `.claude/hooks/extract-experience.sh` 两个文件。不得修改其它 `.claude/hooks/*` 或 `scripts/*`。

## 开发笔记

- 主进程必须先 `cat` 读完 stdin，**禁止**用 `"async": true`（macOS 空 stdin bug #38162）
- worker 调用必须 `nohup ... &; disown`（绕 SessionEnd 后台被杀 bug #41577）
- 任何异常路径都 `exit 0`，不阻塞用户
